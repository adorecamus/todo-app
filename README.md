# Todo App
투두 앱 REST API 서버    

> ### 개발 환경
>     Kotlin / Spring Boot(3.2.0) / Spring Data JPA / PostgreSQL

## 요구사항
- 할 일을 작성합니다.
- 선택한 할 일을 조회합니다.
- 할 일 목록을 조회합니다.
- 선택한 할 일을 수정합니다.
- 선택한 할 일을 삭제합니다.
